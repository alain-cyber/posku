How it works
Three steps
  1. Define SKUs — Each product variant gets a code built from category, product, and up to two attributes. A sequence number ensures uniqueness.
  2. Pick from library — Build a purchase order by selecting saved SKUs, choosing quantities, and confirming unit cost.
  3. Send or print — Export the PO as CSV or print it. Stock-on-hand updates when you mark a PO received.
Storage
All data lives in localStorage on this device. Export to CSV before clearing your browser data.
